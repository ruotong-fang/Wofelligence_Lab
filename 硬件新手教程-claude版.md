## 一、电路基础概念

### 欧姆定律（最重要的公式）

```
V = I × R
电压（V）= 电流（A）× 电阻（Ω）
```

- **电压（V）**：推动电流流动的"压力"，单位伏特
- **电流（I）**：单位时间内通过的电荷量，单位安培
- **电阻（R）**：阻碍电流流动的程度，单位欧姆

### 功率

```
P = V × I
功率（W）= 电压（V）× 电流（A）
```

用于计算元器件发热量和电源需求。例如：5V、500mA 的设备耗电 2.5W。

### 串联与并联

| 类型 | 特征 | 公式 | 典型应用 |
|------|------|------|----------|
| 串联 | 电流相同，电压按比例分配 | R总 = R₁ + R₂ + R₃ | LED限流、分压电路 |
| 并联 | 电压相同，电流按比例分配 | 1/R总 = 1/R₁ + 1/R₂ | 家用电器、多路供电 |

### 数字信号与模拟信号

- **数字信号**：只有高/低两种电平（0 或 1），3.3V 或 5V 为高电平，0V 为低电平。微控制器的基本语言。
- **模拟信号**：连续变化的电压值。传感器（温度、光线、声音）通常输出模拟信号，需经 ADC（模数转换器）转为数字信号。

> **新手建议**：先理解欧姆定律再动手。每次给 LED 串联限流电阻时，都是在实践 V = IR。

## 二、核心元器件

### 基础三件套（必须掌握）

#### 电阻（Resistor）

- 作用：限制电流，保护其他元件
- 单位：欧姆（Ω），常见有 kΩ、MΩ
- 识别：通过色环读取阻值（四环/五环色环法）
- 典型用途：给 LED 串联限流电阻，防止烧毁

```
计算 LED 限流电阻：R = (电源电压 - LED正向电压) / 期望电流
例：(5V - 2V) / 0.02A = 150Ω，选 220Ω
```

#### 电容（Capacitor）

- 作用：存储电荷、滤除电源噪声、稳定电压
- 单位：法拉（F），常用 μF（微法）、nF（纳法）、pF（皮法）
- 使用注意：电解电容有极性，正负不能接反
- 典型用途：在电源引脚旁并联 100nF 去耦电容，是标准操作

#### 二极管 / LED

- 作用：单向导通电流；LED 同时发光
- 识别：长脚为正极（阳极），短脚为负极（阴极）
- 使用注意：**必须串联限流电阻**，否则会烧毁
- 正向压降：普通硅二极管约 0.7V，红色 LED 约 2V，蓝/白 LED 约 3.2V

### 进阶元器件

#### 晶体管（Transistor）

- 作用：电子开关或信号放大
- 类型：NPN 型（常见）和 PNP 型
- 典型用途：用小电流（microcontroller 引脚）控制大电流（电机、继电器）
- 引脚：基极（B）、集电极（C）、发射极（E）

#### 集成电路（IC / 芯片）

- 将大量晶体管和电路集成在一块硅片上
- 使用方法：查找芯片 Datasheet，了解引脚定义和工作条件
- 从运算放大器（LM358）到无线模块（nRF24L01），现代电子的核心

#### 常用传感器模块

| 模块 | 测量对象 | 接口 | 难度 |
|------|----------|------|------|
| DHT11/DHT22 | 温湿度 | 单总线 | ⭐ |
| HC-SR04 | 超声波距离 | GPIO | ⭐ |
| MPU-6050 | 加速度+陀螺仪 | I²C | ⭐⭐ |
| MQ-2/MQ-135 | 气体浓度 | 模拟 | ⭐ |
| MAX30102 | 心率+血氧 | I²C | ⭐⭐ |

## 三、工具清单

### 最小必要工具集（约 ¥200 起步）

#### 面包板（Breadboard）⭐ 优先购买

- 无需焊接，插拔即用，适合快速验证电路原型
- 内部结构：横向两排电源轨（整行导通），中间按列导通（每列5孔）
- 搭配跳线使用，新手最重要的实验工具

#### 万用表（Multimeter）⭐ 优先购买

- 功能：测量电压、电流、电阻、二极管通断
- 推荐型号：胜利 VC890D（¥80-120）、优利德 UT61E（¥150+）
- 排查电路故障不可缺少，贯穿整个学习过程

#### 电烙铁（Soldering Iron）⭐ 优先购买

- 用于将元件焊接到 PCB，固化电路连接
- 推荐控温烙铁：白光 936（¥100）、Hakko FX-888D（¥500+）
- 操作要点：350°C 左右焊接，烙铁头保持镀锡，注意安全

### 进阶工具

#### 可调直流电源

- 提供稳定、可调节的电压和限流保护
- 比用电池方便，带短路保护，调试更安全
- 初期可用 USB 供电模块替代

#### 逻辑分析仪

- 捕获并分析数字信号时序（SPI、I²C、UART 等协议）
- 入门推荐：Saleae 兼容款（¥30-80），配合 PulseView 软件使用

#### 示波器（Oscilloscope）

- 可视化信号波形，调试模拟和数字电路的专业工具
- 入门推荐：Rigol DS1054Z（4通道，¥1500）或 USB 示波器
- 学习 I²C、SPI 等通信协议调试时必备

### 软件工具

| 工具 | 用途 | 价格 |
|------|------|------|
| 立创 EDA | 原理图 + PCB 设计（中文/云端） | 免费 |
| KiCad | 原理图 + PCB 设计（开源） | 免费 |
| Arduino IDE | Arduino 编程 | 免费 |
| Thonny / VS Code | MicroPython 编程 | 免费 |
| Altium Designer | 专业 PCB 设计 | 付费 |
| Fritzing | 面包板电路图绘制（适合新手展示） | 免费/付费 |

## 四、开发平台选择

### Arduino —— 最推荐入门

- **语言**：C/C++（简化版）
- **优点**：生态最成熟，教程资源极丰富，社区活跃
- **常用型号**：Uno（入门标准）、Nano（体积小）、Mega（引脚多）
- **电压**：5V GPIO，兼容大多数传感器模块
- **适合项目**：传感器读取、舵机控制、显示屏驱动、简单自动化

### ESP32 / ESP8266 —— 性价比之王

- **语言**：Arduino C++ 或 MicroPython
- **优点**：内置 WiFi 和蓝牙（ESP32），价格极低（¥15-30）
- **ESP32 特点**：双核 240MHz，内置 ADC/DAC，支持 BLE
- **适合项目**：IoT 物联网、无线数据传输、智能家居、穿戴设备

### Raspberry Pi —— 微型 Linux 计算机

- **语言**：Python（主流）、C、Node.js 等
- **优点**：完整 Linux 系统，可运行复杂算法，有 USB/HDMI/以太网
- **GPIO**：3.3V，注意不能直接接 5V 信号
- **适合项目**：计算机视觉、音频处理、边缘推理、数据记录服务器

> 与情感计算 / 行为研究结合：Raspberry Pi + 摄像头 + 麦克风阵列 + Python/OpenCV，可快速搭建数据采集原型。

### STM32 —— 工业级微控制器

- **语言**：C/C++（HAL 库或寄存器操作）
- **优点**：ARM Cortex-M 内核，性能强、功耗低、实时性好
- **学习曲线**：较陡，需理解时钟配置、中断、DMA 等概念
- **适合项目**：实时控制系统、量产产品、低功耗穿戴设备

### 选择建议

```
完全新手           → Arduino Uno
想做 IoT/无线     → ESP32
需要运行 Python   → Raspberry Pi
有嵌入式基础      → STM32
```

## 五、学习路径

### 第一阶段：理解基础（1-2 周）

- 掌握欧姆定律、串并联、数字/模拟信号的区别
- 实验：用面包板点亮 LED，计算限流电阻值
- 学习资源：Bilibili 电子入门视频、《电子制作入门》

### 第二阶段：入门 Arduino（2-4 周）

按以下顺序完成项目，每个都要自己写代码：

1. LED 闪烁（Blink）
2. 按键控制 LED 开关
3. PWM 调光（模拟呼吸灯）
4. 读取传感器数值（温度/光线）并串口打印
5. 控制舵机转动角度
6. 用 I²C 驱动 OLED 显示屏

> 关键原则：不要只抄代码，每个参数都要知道为什么这样写。

### 第三阶段：学会看数据手册（第 4-6 周）

- 遇到新芯片、新模块，第一步找 Datasheet
- 重点阅读：引脚定义（Pinout）、工作电压范围、典型应用电路
- 练习：拿 DHT22 或 MPU-6050 的 Datasheet，尝试不依赖教程独立接线

### 第四阶段：完成一个完整项目（第 6-12 周）

选择一个与自身兴趣结合的项目，要求具备：

- 传感器输入（至少一路）
- 数据处理逻辑
- 输出或记录（显示、串口、存储卡、网络上传均可）

**项目建议**（结合心理/行为研究方向）：
- 表情/声音数据采集装置（Raspberry Pi + 摄像头 + 麦克风）
- 生理信号记录仪（ESP32 + MAX30102 心率传感器 + SD卡）
- 眼动/头动数据记录器（MPU-6050 + 蓝牙传输到电脑）

### 第五阶段：从面包板到 PCB（第 3 个月+）

1. 用立创 EDA 绘制原理图
2. 根据原理图完成 PCB 布线
3. 在立创商城打样（5 块约 ¥5-30）
4. 焊接元件，上电测试

这一步让你真正理解从"能用"到"好用"之间的工程差距。
