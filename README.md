## 🌌 操作哲学 · Operating Philosophy

1. **先发射，再优化。** 残缺的仓库胜过完美的计划。
   **Launch first, optimize later.** A messy repo beats a perfect plan that never ships.

2. **好奇心是唯一用不完的燃料。**
   **Curiosity is the only fuel that doesn't run out.** (Coffee is a close second.)

3. **每一条死路都是重新定向的轨迹。**
   **Every dead end is just a redirected trajectory.** (This is what we tell ourselves.)

4. **最好的研究发生在不该相交的事物之间。**
   **The best research happens at intersections.** Psychology × AI × what-was-I-thinking.

5. **没有弯路，只有还没走完的必经之路。**
   **There are no detours. Only roads you didn't know you needed.** Confusion × Clarity × oh-so-that's-why.
